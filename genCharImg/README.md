## 实现方案

实现字符画是简单的, 我们只需要找到一组大小不同的字符作为其骨架, 然后将图片转换成灰度图后再把每个像素点按照灰度值和字符大小的对应关系映射成字符矩阵即可. 

这里我们使用

```python
charset = {"@", "#", "*", "o", "+", "-", "."}
```

作为骨架. 

我们使用 gamma 校正的rgb -> 灰度值算法, 其公式如下

```python
gamma = 2.2
gray = pow((r ** gamma + (1.5 * g) ** gamma + (0.6 * b) ** gamma)
           /  (1 + 1.5 ** gamma + 0.6 * gamma), 1. / 2.2)
```
